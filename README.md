# Java Macera Oyunu

##  Proje Açıklaması
Bu proje, Java programlama dili ile geliştirilmiş metin tabanlı bir macera oyunudur. Oyuncu, farklı senaryolar arasında seçim yaparak oyunun gidişatını belirler.

##  Oyun Mantığı
Oyuncu karşısına çıkan durumlarda seçim yapar ve bu seçimlere göre farklı sonuçlara ulaşır. Oyun tamamen kullanıcı etkileşimine dayalıdır.

##  Özellikler
- Seçim tabanlı oyun akışı
- Farklı senaryo dallanmaları
- Kullanıcı etkileşimi
- Basit metin tabanlı yapı

##  Kullanılan Teknolojiler
- Java

##  Amaç
Koşul yapıları (if-else, switch-case) ve mantıksal akış geliştirmeyi öğrenmek.

##  Öğrendiklerim
- Koşul yapıları
- Kullanıcı girişleri
- Oyun mantığı oluşturma
- Akış kontrolü
